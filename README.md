# CYB333 Port Scanner Project

## Project Overview

This project is a Python automated port and vulnerability scanner created for CYB 333 Security Automation. The purpose of the project is to demonstrate how Python can be used to automate basic security testing by scanning a target host for common open ports and identifying possible security concerns.

The scanner checks a list of common ports, identifies the service normally associated with each port, and provides a basic security note when an open port is discovered. The tool also creates a scan report that can be reviewed after the scan is complete.

## Project Objectives

The main objectives of this project are:

- Create a Python script that scans common network ports
- Identify open ports on a target system
- Match open ports with common services
- Provide basic security notes for possible vulnerabilities
- Generate a text-based scan report
- Demonstrate security automation concepts using Python

## Files Included

- `Scanner.py` - Main Python script for scanning ports
- `sample_scan_report.txt` - Example scan report
